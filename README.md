# Agritech App – README.md
## Présentation

### Agritech App est une application moderne dédiée à l’agriculture intelligente.
Elle permet de visualiser les données agricoles, suivre des indicateurs, analyser les performances et fournir des statistiques précises pour aider les agriculteurs, organisations et décideurs.

### L’objectif est d’utiliser la technologie pour améliorer la gestion agricole et la prise de décision.

# Fonctionnalités principales

- Dashboard interactif (indicateurs FAO, Formation, marchés, météo…)

- Gestion des données agricoles (cultures, stocks, régions…)
